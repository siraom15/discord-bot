# Bot Discord
## วิธีลง Bot ในเครื่อง
> สิ่งที่ต้องมี NodeJS version > 12

## ขั้นตอนการลง
* clone/download repository นี้
* แตกไฟล์
* Run " $ npm install " ใน command line 
* ตั้งค่า Token bot discord ในไฟล์ config.json

## วิธีใช้งาน
### Run file runbot.bat
### cd src node app.js

## หรือต้องการเชิญ Bot เข้า Discord
